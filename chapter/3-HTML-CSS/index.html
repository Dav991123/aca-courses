How The Website Works

3 Mots important things

HTML is a markup language and CSS is a styling language. JavaScript, however, is a programming language. Hence, they are all web languages, but they perform different jobs

HTML

  • HyperText Markup Language
  • NOT a programing language
  • HTML is the standard markup language for creating Web pages
  • HTML describes the structure of a Web page
  • HTML elements tell the browser how to display the content
  • Everything Should Be Made as Simple as Possible, But Not Simpler

HTML Page Structure

Doctype

The declaration is not an HTML tag. It is an "information" to the browser about what document type to expect.

HTML Syntax

  • Elements names surrounded by angle brackets
  • Normally come in pairs (start tan and end tag)
  • End tag is the same but with a forward
  • Some tags close themselves (Remnant of XHTML)

Tag Attributes

  • All tags can have attributes
  • Provide information about an element
  • Placed within the start tag

Inline .vs Block Level Elements

  • Inline Elements
    • Do not start on a new line
    • Take only the necessary width
  • Block Elements
    • Start on a new line
    • Take full width available
  • Block Tag: div, h1 - h6, p, form, header, ...
  • Inline Tag: span, img, a, input, ...
