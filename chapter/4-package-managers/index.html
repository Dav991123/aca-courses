nvm Node Version Manager

  • nvm install node # "node" is an alias for the latest version
  • nvm install 6.14.4 # or 10.10.0, 8.9.1, etc
  • nvm use node
  • nvm uninstall v14.15.4
  • nvm alias default 6 # Set default node version
